# IMG-AMRITA: Indian Music Genre Classification Dataset

## Overview

IMG-AMRITA is a publicly available dataset developed for research in Indian Music Genre Classification using machine learning and deep learning techniques. The dataset consists of mel-spectrogram images generated from 30-second audio recordings belonging to diverse Indian music genres.

The dataset is designed to facilitate research in music information retrieval, audio signal processing, computer audition, and deep learning-based music classification systems.

## Dataset Description

* Dataset Name: IMG-AMRITA
* Data Representation: Mel-Spectrogram Images
* Image Format: PNG
* Audio Duration: 30 seconds per sample
* Application: Indian Music Genre Classification
* Total Dataset Size: 332 MB

Each audio recording is converted into a mel-spectrogram image, which provides a time-frequency representation of the musical content. These spectrogram images can be directly utilized as input to convolutional neural networks (CNNs), vision transformers, and other deep learning architectures for genre classification tasks.

## Genres Included

The dataset contains samples from the following Indian music genres:

1. Baul
2. Bihu
3. Carnatic
4. Cinematic
5. Ghazal
6. Hindustani
7. Kathakali
8. Punjabi
9. Rabindra Sangeet
10. Rajasthani
11. Sopana Sangeetham
12. Sufi

## Mel-Spectrogram Generation

The original audio recordings were segmented into 30-second clips and transformed into mel-spectrogram representations. Mel-spectrograms provide a perceptually meaningful frequency representation by mapping frequencies onto the mel scale, which closely approximates human auditory perception.

These spectrogram images preserve both temporal and spectral characteristics of the music, making them suitable for deep learning-based classification frameworks.
